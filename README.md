# Narmada Parikrama

A dedicated digital platform built for the **Maa Narmada Parikrama Vasi Seva Samiti** to support pilgrims undertaking the sacred Narmada Parikrama journey.

🌐 Live Website: https://narmada-parikrama.vercel.app

---

## About

Narmada Parikrama is one of India's most revered spiritual pilgrimages, where devotees undertake a complete circumambulation of the holy River Narmada. This platform was developed to help the Narmada Parikrama Trust provide pilgrims with easy access to information, resources, and support services through a modern and accessible web experience.

The project aims to strengthen the Trust's digital presence while making important information readily available to devotees, volunteers, and visitors.

---

## Local Development

### Clone the Repository

```bash
git clone https://github.com/Darshil-Sharma/narmada-parikrama.git
```

### Navigate to the Project Directory

```bash
cd narmada-parikrama
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python main.py
```
